# MODEL CARD REPORT

## Google Gemma 4 12B IT

### 1. Introduction

Gemma 4 12B IT is an advanced open-weight Artificial Intelligence model developed by Google DeepMind. The model is designed for instruction-following tasks and can understand and generate human-like responses. It supports multimodal inputs such as text, images, and audio, making it suitable for a wide range of AI applications.

| Architecture     | Decoder-Only Transformer   |
| License          | Apache 2.0                 |

---

### 3. Model Architecture

Gemma 4 12B IT is built using a Transformer-based architecture optimized for efficient reasoning and language understanding. The model uses advanced attention mechanisms and positional encoding techniques to process long documents and maintain contextual understanding across large inputs.

Key architectural features include:

* Decoder-only Transformer architecture
* Hybrid attention mechanism
* Proportional Rotary Positional Embeddings (p-RoPE)
* Long-context processing capability
* Multimodal input support

---

### 4. Training Dataset

The model was trained using a large-scale multimodal dataset collected from diverse sources. Training data includes:

* Public web documents
* Programming code repositories
* Mathematical and scientific content
* Educational resources
* Images and visual data
* Audio data
* Multilingual content from over 140 languages

This diverse dataset helps the model perform effectively across various domains and tasks.

---

### 5. Training Methodology

The training process consisted of multiple stages:

#### Pre-Training

The model was trained on large volumes of text, code, image, and audio data to learn language patterns, reasoning abilities, and knowledge representation.

#### Instruction Tuning

After pre-training, the model was fine-tuned using instruction-based datasets to improve its ability to follow user commands and generate useful responses.

#### Safety Alignment

Additional safety and evaluation procedures were applied to reduce harmful outputs and improve reliability.
